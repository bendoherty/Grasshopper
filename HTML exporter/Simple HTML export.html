This setup allows for GH data to be written into an HTML file.
It isn't the most elegant solution, but it does minimise external dependencies.
veryBasicHTMLtemplate.gh

You make your page in the blue boxes, and the yellow boxes are more or less fixed.
It comes with JQuery and Bootstrap from a CDN, so there's no installing needed to get nice responsive layouts.


The imgName in the component maps to the variable of the same name in the HTML template by using python's locals() feature. This means that you just need to add the name to the component and it becomes available in the HTML.



There is also a very rough image grab tool that puts the image in the page. This needs much more updating to support cameras etc.

Calling this code:
import rhinoscriptsyntax as rs
rs.SendKeystrokes('-viewcapturetofile')
rs.SendKeystrokes('C:\\Users\\bdoherty\\Desktop\\testghwebsite\\'+str(x)+'.jpg')
rs.SendKeystrokes('')
